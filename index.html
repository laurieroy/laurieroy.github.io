Pizza Ordering site

The assignment was to make a form for the customer to order a pizza, give method of payment, and address to be delivered. This data is quality checked, and if passes, form data is passed through to a database. From the database, workers would build the pizza, and mark completed when done. This updates a gridview the manager would have of how many orders are in the works. Data is separated into concerns.

File Transfer App

Desktop app checks a folder for new/modified files and transfers new files from one folder to another. Time of last transfer is recorded in a database. User can specify the type, age of file, directory location. GUI built with Tkinter.

This project is the culmination of several assignments, each increasing in complexity. I encountered several hurdles, from confusing myself by making timezones more complicated than they are, and used some functions that didn't scale well. I dove into the documentation and checked online forums to figure it out.
